Basic Information Tab

This tab allows you to specify some basic information about the computer's properties.

  1. Computer Name - Specify the name of this computer as it appears on the network.

    Requires reboot to take effect!

  2. Domain - Specify the domain for this computer to join.

    Requires a domain account with permission to join the domain!

  3. Time Zone - Specify the time zone to set for this computer.
  4. Primary NTP Server - Either select from the list or manually specify the primary NTP (network time protocol) server to use for syncing the time of the computer.
  5. Perform Time Server Sync? - When checked, specifies that the program should perform an initial time server sync using the time server specified by the Primary NTP Server.

Zest Script Documentation
=======================================

Zest Script is an all-in-one configuration manager for IT companies and MSP's that enable them to on-board client's computers
quickly and easily using a configurable file. It stores the configuration for a company or computer in a single .XML file that can
later be read by the application and then modified for the specific computer that is being on-boarded.

Feature Overview
=======================================
* Customize the computer name, domain, time-zone, and NTP server.
* Create user accounts.
* Create pre-configured Wi-Fi and VPN profiles.
* Add a list of applications to install.
* Create network drive mappings.
* Create printer mappings.
* Import/Export configuration file for later use.
* View a configuration file without messing up the current configuration.
